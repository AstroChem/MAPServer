MAPS Runs

This page catalogs all imaging attempts carried out on all measurement sets of the MAPS large program. The run_id is a unique primary key assigned to every new invocation of an imaging script to a particular disk / transition / measurement set combination. If you would like to refer precisely and uniquely to the products of a particular imaging attempt, quoting the run_id is the best way to do it. You can click on the run_id in the table below to be taken to the summary page for that run. Alternatively, if you already know the run_id, you can navigate to the summary page directly by substituting it in the address like so: thiswebsite.com/runs/run_id/
