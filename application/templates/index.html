MAPS Image Products

Welcome to the MAPS image products portal.

How to use this website

This website is designed to visualize the imaging products for every component of the MAPS dataset, by providing different views of the results based upon the criterion you are looking for. The easiest way to get started is to click around using the navigation bar above. You browse the available products by disk or transition, and then drill down into the results from various imaging algorithms.

If you have suggestions on how to improve these webpages, please contact Ian Czekala iczekala@berkeley.edu.
